What is a cookie?

They are text files recording certain actions such as:

  • The selected language
  • When the user logs in/out
  • The number of times an user comes back
  • If the user agrees with certin terms and conditions

For what purpose?

In the case of this website it will serve 3 purposes:

  • To know, of course, if the user agrees with the terms of cookie usage
  • To remember what is the language chosen by the user
  • To monitor, thanks to Google Analytics, the traffic on my portfolio
